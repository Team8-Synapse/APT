# Amrita Placement Tracker (APT)

## Overview
Amrita Placement Tracker is a comprehensive web application designed to streamline the placement process at Amrita Vishwa Vidyapeetham. It connects students, placement coordinators (CIR), and alumni to ensure a smooth transition from campus to corporate.

## Key Features
### For Students:
- **Dashboard**: View eligible drives, upcoming events, and application status.
- **Profile Management**: Maintain academic records, resume, and skills.
- **PrepHub**: Access interview materials, roadmaps, and cheat sheets.
- **Alumni Insights**: Read success stories and interview experiences.
- **One-Click Apply**: Apply to companies directly through the portal.

### For Admins (CIR):
- **Drive Management**: Create and manage placement drives.
- **Student Database**: Access student profiles and track eligibility.
- **Analytics**: Visualization of placement statistics (Placed vs. Unplaced).
- **Announcements**: Broadcast important updates to students.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS, Vite.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB.
- **Authentication**: JWT & Bcrypt.
