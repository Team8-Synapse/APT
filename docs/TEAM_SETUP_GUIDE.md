# Team Setup & Onboarding Guide

## Introduction
Welcome to the Amrita Placement Tracker development team! This guide ensures you have access to everything needed to start contributing.

## Roles & Responsibilities
- **Frontend Developer**: manages React components, UI/UX, and state management.
- **Backend Developer**: Manage API endpoints, database schemas, and server logic.
- **Full Stack Developer**: Bridges the gap, handling integration and end-to-end features.
- **Project Lead**: Oversees architecture, code reviews, and timeline.

## Access Requirements
1. **GitHub Repository**: Request access from the maintainer.
2. **MongoDB Atlas**: Ask for the connection string or get invited to the organization.
3. **Environment Keys**: Secure a copy of the `.env` file from the team lead.

## Communication
- **Updates**: Daily standups via WhatsApp/Discord.
- **Code Review**: All PRs must be reviewed within 24 hours.

## Best Practices
- **Never commit .env files.**
- **Always pull from main before starting a new feature.**
- **Document complex functions.**
